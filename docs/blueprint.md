# **App Name**: Ceylon Velo Luxe

## Core Features:

- Island Curator AI Tool: An intelligent travel planning tool that generates bespoke itineraries based on guest preferences for wildlife, relaxation, or cultural depth.
- Destination Explorer Grid: An interactive, visual-heavy showcase of Sri Lankan regions including Mirissa, Ella, and Sigiriya with hover-revealed details.
- Bespoke Villa Inquiry System: A lead-capture system allowing users to request bookings for hand-picked private retreats, saving entries to the inquiry database.
- Experience Three Pillars: A focused layout highlighting signature services: Boutique Villas, Signature Tours, and Authentic Cuisine.
- Heritage Client Stories: An editorial-style testimonial slider featuring social proof from luxury travelers.
- Visual Diary Wall: A minimalist gallery component integrating island imagery with smooth transition effects to emulate an Instagram feed.
- Animated Journey CTA: A high-conversion footer section featuring atmospheric background visuals and call-to-action motion effects.

## Style Guidelines:

- Primary Color: Deep Forest Green (#1F5145), evoking the lush landscapes of the central highlands.
- Background Color: Silk Parchment (#F8F5EF), providing a warm, high-end editorial feel rather than sterile white.
- Accent Color: Sri Lankan Amber (#E99A3D), used sparingly for buttons and active UI highlights for energy.
- Headline Font: 'Playfair' (Serif) for a sophisticated, editorial magazine look; Body Font: 'Inter' (Sans-serif) for clean readability. Note: currently only Google Fonts are supported.
- Elegant, ultra-thin line icons that maintain the minimalist and airy aesthetic of the project.
- A spacious, editorial-magazine layout with significant white space, wide gutters, and large full-bleed imagery.
- Subtle parallax scrolling and fade-up reveals for sections, utilizing Framer Motion for premium fluidity.
